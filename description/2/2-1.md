## 1. 호출 스택 - 예제 1

```javascript

function first() {
    second();
    console.log('첫 번째');
}

function second() {
    third();
    console.log('두 번째');
}

function third() {
    console.log('세 번째');
}

first();

```

## 2. 호출 스택(+이벤트 루프) - 예제 2

<br>


```javascript

function run() {
    console.log('3초 후 실행');
}

console.log('시작');
setTimeout(run, 3000);  // 비동기
console.log('끝');

```

<br>

    - 시작 - 끝 - 3초 후 실행 순서대로 결과가 나옴 
    - 호출 스택으로만으로는 설명이 안됨
    - 호출 스택 + 이벤트 루프로 설명 가능함
